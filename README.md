The purpose of this project is to solve real-world business questions using SQL queries and analytical thinking.

## Business Questions

### Customer Analysis
1. How many customers are from each country?  
2. How many orders has each customer placed?  
3. Which customers have the highest average order value?  
4. Which customers placed orders only in the first year of operations?  

### Product Analysis
5. What is the top-selling product in each category based on total quantity sold?  
6. Which products were sold fewer than 20 times in each year?  
7. What is the total revenue generated by each product?  

### Revenue / Region Analysis
8. What are the top three highest-revenue products in each country/region?  
9. What is the monthly sales revenue trend over time?  
10. Which product categories generate the highest total revenue?
